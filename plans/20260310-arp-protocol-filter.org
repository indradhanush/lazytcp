#+TITLE: lazytcp ARP Protocol Filter Support

* Objective
Ensure ARP packets parsed from pcap are represented in app state so the Protocol filter can show and filter =arp=.

* Requested Behavior
- When pcap contains ARP packets, Protocol popup includes =arp=.
- Selecting =arp= filters packet list to ARP packets.

* Scope
In scope:
- Extend tcpdump line parsing to include ARP lines.
- Add parser tests for ARP request/reply lines.
- Add app tests for protocol candidates/filtering with ARP packets.

Out of scope:
- Non-ARP link-layer protocol expansions.
- Capture backend process changes.

* Implementation Plan
1. Add ARP parsing branch in =src/api.rs= and keep existing IP/IP6 behavior intact.

3. Add app-level tests for protocol popup candidate inclusion and ARP filtering.
